# Jr Lions Lacrosse — Community Savings Book HQ

**Live Campaign Dashboard** for the Jr Lions Lacrosse Community Savings Book fundraising program.

---

## What This Is

This repository hosts the complete campaign command center for the **Jr Lions Community Savings Book** — a $30 digital savings pass offering $500+ in local offers at Hunterdon County golf courses, restaurants, car washes, simulators, and family destinations. All proceeds support Jr Lions Lacrosse.

---

## Accessing the Dashboard

Once GitHub Pages is enabled, the dashboard is live at:

```
https://YOUR_USERNAME.github.io/jr-lions-savings-book/
```

Open `index.html` in any browser to use it locally without deploying.

---

| File | Description |

## Dashboard Tabs

- **Overview** — Timing analysis, KPIs, campaign files, one-sentence pitch
- **Action Steps** — 32 checkable steps across 6 phases (saves progress in browser)
- **Businesses** — All 28 target businesses, filterable by category
- **Email Templates** — 10 copy-ready templates including 4 personalized anchor emails
- **Marketing** — Social calendar, distribution setup, promotion strategy
- **Financial** — Scenario model, cost breakdown, value stack
- **GitHub Setup** — Step-by-step instructions to deploy this dashboard live

## Deploying to GitHub Pages

1. Go to your repository → **Settings** → **Pages**
2. Source: **Deploy from a branch**
3. Branch: **main** | Folder: **/ (root)**
4. Click **Save**

Your site will be live within 1–3 minutes at the URL above.
