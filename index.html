Timing Analysis — Is End of June Too Late to Launch?
Verdict
Launch July 1st. Don't Wait.
End of June is not too late. Summer is actually the strongest season for every offer category in this book.
The concern (valid but manageable)
  • The spring lacrosse season ends late May/early June — the captive "team parent" audience will scatter somewhat.
  • Launching at the close of a season risks feeling like an afterthought tied to a year that just closed.
Why summer works better than you think
  • Golf is at absolute peak June–September. Your highest-value coupons are maximally relevant right now.
  • Families have more time — ice cream, mini golf, dining, and bowling all spike in summer.
  • Car washes spike in summer too. Every offer category aligns perfectly with the season.
  • This is a social-media-driven campaign, not player-selling. Your audience is the whole community, not just current team parents. It doesn't shrink when practice ends.
  • Digital book means no print-run risk. You can sell through September, October, and beyond without waste or cost.
What NOT to do
  • Don't brand it "support our spring season." Spring is over. Brand it "support Jr Lions year-round."
  • Don't wait for fall to get a "clean start." You'd forfeit 3 months of prime golf/summer selling time.
Recommended path
  • Now–mid June: Secure business commitments and build digital assets.
  • July 1: Hard launch. "Jr Lions Community Savings Book — Summer Edition."
  • July–August: Active selling via social media and community channels.
  • September: Second push as back-to-school energy hits. Reframe as "year-round savings."
  • Book valid through June 30, 2027 — every buyer gets a full year of value regardless of when they buy.

Digital Distribution Setup
  • Payment: Gumroad (easiest) or Stripe — buyer pays $30, receives download link automatically. Zero manual work.
  • Delivery: PDF of the digital coupon book, OR a password-protected Google Drive folder with individual coupon cards.
  • QR codes: Each business gets a unique QR. Buyers show on their phone. Staff scans to validate. Free tool: qr-code-generator.com
  • Hub link: Create a Linktree or single landing page. Point all social bios here. Everything lives in one URL.
  • Email list: Collect buyer emails via Gumroad or a Google Form. Use free Mailchimp tier to send updates and partner spotlights.
Promotion (Program-Driven)
  • Partner reshares: Ask each confirmed business to share the launch post. Their followers are your target market. This is free and powerful.
  • Community groups: Post in Hunterdon County Facebook groups, Nextdoor, school parent groups. These convert well.
  • Paid ads: $50–$150 on Facebook/Instagram, target Hunterdon County, ages 25–55. The $30 price converts easily.
  • Local press: Email Hunterdon Democrat and TapInto Flemington. A short community pitch gets free coverage. Great for credibility.
  • Email: Every buyer gets a "share with a friend" message. Your best growth lever is word-of-mouth from happy buyers.

GitHub Pages Deployment

What you get when done

A permanent URL you can share with co-directors, board members, and volunteers. This dashboard plus all downloadable files will live at one link forever. Updates you push to GitHub appear live in under a minute.

1

Install Git

Mac: open Terminal, run xcode-select --install. Windows: download from git-scm.com and install with defaults.

2

Create a free GitHub account

Go to github.com → Sign up → Free plan. Your username becomes part of the site URL.

3

Create a new repository

On GitHub → green "New" button → Name: jr-lions-savings-book → Set to Public → click "Create repository."

4

Build your local folder

Create a folder on your computer with this exact structure:

jr-lions-savings-book/ ├── index.html ← this dashboard ├── README.md └── files/ ├── Jr_Lions_Community_Savings_Book_Playbook.docx ├── Business_Outreach_Tracker.xlsx └── Campaign_Financial_Dashboard.xlsx
5

Push files to GitHub

Open Terminal in your folder and run these commands one at a time:

git init git add . git commit -m "Initial commit — Jr Lions HQ" git branch -M main git remote add origin https://github.com/YOUR_USERNAME/jr-lions-savings-book.git git push -u origin main
6

Enable GitHub Pages

On GitHub → your repo → Settings tab → Pages (left sidebar) → Source: "Deploy from a branch" → Branch: main → Folder: / (root) → Save.

7

Your live URL

https://YOUR_USERNAME.github.io/jr-lions-savings-book/

Goes live in 1–3 minutes. Bookmark it and share with your team.

8

Pushing future updates

Whenever you update any file, save it in the folder and run:

git add . git commit -m "Update [describe what changed]" git push

Live within 30 seconds of pushing.
